# Formcord Website

This is the repository for the official website and documentation of **[Formcord](https://formcord.vercel.app/)** — a lightweight, universal Discord notifications package.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS
- **Syntax Highlighting**: Shiki

## Running Locally

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Repository Structure
- `app/`: Next.js App Router pages (Home, Docs, Changelog, Sitemap)
- `components/`: UI components (Hero, Docs formatting, Layout)
- `public/`: Static assets
